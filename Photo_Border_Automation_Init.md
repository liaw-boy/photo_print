# 專案初始化指引：相片白邊自動化處理程式

## 專案概述
你現在是一位資深的影像處理與自動化開發專家。我需要你協助我開發一款自動化程式，功能對標 Adobe Lightroom iPad 版的「加上白邊並匯出」功能。
該程式需要能讀取相片，根據設定的比例自動計算並加上白邊，最後輸出高品質的影像。

## 核心需求
1. **白邊計算與裁切邏輯**：
   - 支援按比例添加白邊（例如：基於影像長邊或短邊的特定百分比向外延伸）。
   - 支援目標長寬比填充（例如：將任意比例的照片透過補白邊的方式，轉換成 1:1 或 4:5 等社群媒體常用比例，而不變更原圖比例）。
2. **影像品質與攝影數據保留**：
   - 匯出時必須保持原圖的高解析度與無損畫質。
   - **必須完整保留原始相片的 EXIF 資訊**（包含相機型號、光圈、快門、ISO 等拍攝參數，這對攝影作品極為重要）。
   - 正確處理並保留原始檔案的色彩空間 (Color Space，如 sRGB / Adobe RGB)。
3. **批次自動化處理**：
   - 能夠指定輸入資料夾，對內部所有照片進行批次自動化處理，並輸出到指定的資料夾中。
   - 具備簡單易用的執行方式（例如命令列 CLI 參數設計）。
4. **技術與架構偏好**：
   - 專案須基於開源、免費的生態系與工具進行開發（例如 Python 搭配 Pillow / ImageMagick 等）。
   - 程式碼應具備良好的模組化設計，以便未來能輕鬆擴充其他影像處理模組（例如浮水印添加、套用 LUT 濾鏡等功能）。

## 你的初始任務
請仔細閱讀以上需求，並提供以下內容來啟動我們的專案：
1. 針對此需求，推薦最適用的開發語言與開源影像處理函式庫，並簡述你的選擇原因。
2. 規劃這個自動化程式的基礎架構與核心執行流程。
3. 撰寫一個初步的 MVP (最小可行性產品) 程式碼範例，重點示範「如何計算並加上白邊」以及「如何在另存新檔時無損轉移 EXIF 資料」。

請全程以繁體中文與我進行後續的技術討論與開發指導。
